浏览器内核简介

Rendering Engine可以翻译为”渲染引擎”,一般称之为”浏览器内核”。主要用来解释html，css,js并且将效果呈现在网页上。不同的内核对网页的解释不同，因此呈现的效果也不同。下面就简单介绍一下主流的浏览器内核。主要有：Trident,Gecko,Presto,Webkit,Blink。

  • 内核分类
    • Trident
      代表 IE
    • Gecko
      代表Mozilla Firefox
    • Presto
      代表 Opera(已废)
    • Webkit
      代表 Safari
    • Blink
      代表 Chromium

Meteor简介

Meteor——JavaScript应用平台，创建让你乐于使用的app,其运行速度超乎你的想象。

Meteor是一个完全开源的平台，使用纯javascript编写web和mobile应用

愉悦的用户体验

现代化的用户界面
使得应用就像Facebook和Twitter一样高级，或者想一个完美的桌面应用，而不是由链接拼接的一个个页面

浏览器和移动并行
桌面、手机或者平板完美体验

实时更新
数据改变即更新，用户无缝合作

极速响应
用户变更，立即可见。没有等待服务器响应的缓慢感受

快速、有趣的开发

极少的代码量
10行代码完成别的需要1000行代码才能完成的工作。归功于继承从数据库到用户屏幕的一站式的响应式编程模型

一种语言，从前到后
同种形式的API意味着，使用一样的语言来编写代码，同时运行在客户端和服务器端

统一的包管理系统
浏览器，服务器端和移动设备享有同一包管理系统，一个构建工具就可以构建整个应用

热部署>
一个命令就能将应用推送到生产环境，并且更新所有连接的浏览器和相关设备，完全不需要再经过应用商店

详情见meteor官网

Linux常用命令

简单命令

  1. ls - List
    ls 会列举出当前工作目录的内容（文件或文件夹）

  2. mkdir -Make Directory
    新建一个目录

  3. pwd -Print Working Directory
    显示当前工作目录

  4. cd - Change Directory
    切换到指定的工作目录

  5. rmdir - Remove Directory
    删除给定的目录

  6. rm - Remove
    删除给定的文件或文件夹

  7. cp - Copy
    复制文件或文件夹

  8. mv - Move
    移动文件或文件夹，重命名文件或文件夹

  9. cat - concatenate and print files
    在标准输出（监控器或屏幕）上查看文件内容

  10. tail - print TAIL(from last)
    从文件末尾输出

  11. less - print LESS
    按页或按窗口打印文件内容

  12. grep
    在给定的文件中搜寻指定的字符串

  13. find
    在给定位置搜寻与条件匹配的文件

  14. tar
    创建、查看和提取tar文件

  15. gzip
    创建和提取gzip压缩文件

16.unzip
对gzip文件进行解压

  5. nice
  6. renice
  7. kill
